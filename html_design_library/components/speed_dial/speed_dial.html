Speed Dial

A floating action button that expands into a series of related actions. Designed for primary interactions with a Cyberpunk-Lite aesthetic, featuring glassmorphic action buttons and spring-based animations.

Key Features

  • ■ Glassmorphic action buttons with 12px backdrop blur.
  • ■ Staggered spring animations for action reveal.
  • ■ Multi-directional support (Bottom-Right, Bottom-Left, etc.).
  • ■ Accessibility: ARIA states, keyboard support (Esc/Tab).
