Sidebar Navigation

A professional-grade navigation system with glassmorphic depth, responsive behavior, and mini-mode persistence.

Mini Sidebar Persistence

The sidebar state (full vs mini) is automatically saved to `localStorage`, ensuring a consistent experience across page reloads.

// AEASidebar state saved!
localStorage.getItem('aea-sidebar-mini');

Responsive Transitions

On mobile ( < 768px), the sidebar transforms into an off-canvas drawer with a blurred backdrop.

Resize window to see mobile behavior.

Interaction Events

The sidebar dispatches custom events whenever its state changes. This allows the main content area (like a D3 graph) to recalculate its viewport.
